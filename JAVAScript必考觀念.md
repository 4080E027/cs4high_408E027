# JAVAScript程式格式

### 放在<head>標籤 使用<script>
```
<!DOCTYPE html> 文件類型
<html>建立網頁
  
<head>頭部
<script>頭部
function myFunction() { 自由指派給 變數/屬性  取名變數/屬性
  document.getElementById("demo").innerHTML = "Paragraph changed."; 表示該頁面HTML的內容。
}                                                  是document 的一個方法，會依照參數demo找到id="demo"的HTML元素。指定HTML內容。
</script>尾部
</head>尾部
  
  
  
<body>

<h2>JavaScript in Head</h2>  h1~h7顯示字體大小

<p id="demo">A Paragraph.此p標籤是HTML的已定義元素之一。這個p標籤被定義為“段落”，其中您所寫的任何內容都將被視為網頁上ㄉ的段落。

<button type="button" onclick="myFunction()">Try it</button> button按鈕

</body>
</html> 
```
